## :dart: Features

- [x] Interface Web
- [x] Interface Mobile
- [x] API REST
- [x] Cadastro de aulas
- [x] Listagem de aulas
- [x] Favoritar aulas
- [x] Filtrar aulas por matéria, dia da semana e horário
- [x] Entrar em contato por Whatsapp

## :white_check_mark: Tecnologias
